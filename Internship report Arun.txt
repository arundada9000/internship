# Abstract

This internship report presents the outcomes and experiences gained during a 10-week internship at Sajilo Digital Pvt. Ltd., where I worked as a Frontend Developer. The primary objective of the internship was to gain practical industry exposure and apply academic knowledge to real-world web development projects. During the internship period, I was actively involved in designing and developing the official company website sajilodigital.com.np using Next.js, a modern React-based framework.

The internship focused on developing responsive, performant, and user-friendly web interfaces following industry best practices. Core frontend concepts such as component-based architecture, client-side and server-side rendering, routing, reusable UI components, and performance optimization were implemented throughout the project. The website was designed to represent the company’s digital presence effectively while ensuring accessibility, scalability, and maintainability.

Through this internship, I gained hands-on experience in professional frontend development workflows, version control, project structuring, and collaboration within a development team. The internship significantly enhanced my technical skills, problem-solving abilities, and understanding of modern web technologies. Overall, this internship served as a valuable bridge between academic learning and professional software development practices.

Keywords: Next.js, Frontend Development, React, Responsive Design, Web Development

# Chapter 1: Introduction
1.1 Introduction
In the modern digital era, a professional and well-structured website plays a crucial role in establishing an organization’s online presence and credibility. A company website serves as the primary medium through which users access information about services, vision, and expertise. With the rapid growth of web technologies, modern frameworks such as Next.js have enabled developers to build fast, scalable, and search-engine-optimized web applications.

This project focuses on the design and development of the official website of Sajilo Digital Pvt. Ltd. using Next.js. The website sajilodigital.com.np is developed to represent the company’s digital identity, showcase its services, and provide an engaging and responsive user experience. The project includes multiple sections such as Home, About Us, Services, Portfolio, and Contact, designed with a clean and modern user interface.

The website is developed following component-based architecture, ensuring maintainability and scalability. Responsive design principles are applied to ensure compatibility across various devices such as desktops, tablets, and mobile phones. Performance optimization and structured routing are implemented to enhance user experience and accessibility. This project demonstrates the practical application of modern frontend development techniques to create a real-world, production-ready web application.

# 1.2 Problem Statement

Many growing digital companies face challenges in establishing a strong online presence due to outdated, slow, or poorly structured websites. Without a modern and optimized website, organizations struggle to communicate their services effectively and engage potential clients.

Prior to this project, there was a need for a professional, responsive, and performance-optimized website that could accurately represent Sajilo Digital Pvt. Ltd. and its service offerings. Common problems identified include:

Lack of a modern, scalable web platform built with current technologies.

Poor responsiveness across different screen sizes and devices.

Limited performance and search engine optimization.

Difficulty in maintaining and updating content with traditional static websites.

This project aims to address these issues by developing a modern company website using Next.js that ensures performance, responsiveness, and maintainability.

# 2.5 Literature Review / Related Study

Modern web development has evolved significantly with the introduction of component-based frameworks and performance-oriented technologies. Next.js, built on top of React, provides features such as server-side rendering, static site generation, and optimized routing, which enhance application performance and search engine visibility.

According to Vercel documentation, Next.js improves page load speed and SEO by pre-rendering pages and optimizing assets. Marcotte’s principles of responsive web design emphasize flexible layouts and media queries to ensure compatibility across devices, which were applied in this project.

Research in frontend development highlights that component-based architecture improves code reusability and maintainability, making it suitable for scalable web applications. Clean UI design and optimized user experience significantly impact user engagement and trust in digital platforms, reinforcing the importance of structured and visually appealing company websites.

These studies and practices guided the design and development of the Sajilo Digital Pvt. Ltd. website, ensuring modern standards in performance, responsiveness, and usability.

# 3.3 Description of the Project(s) Involved During Internship

During the internship at Sajilo Digital Pvt. Ltd., I worked on multiple projects that enhanced my practical skills in frontend development and basic backend/network-based systems. These projects involved the use of Next.js, React, and Python, and provided exposure to real-world problem-solving, performance considerations, and user-focused design. The major projects undertaken during the internship are described below.

Project 1: Company Website – Sajilo Digital Pvt. Ltd.

The primary project during the internship was the design and development of the official website of Sajilo Digital Pvt. Ltd. using Next.js. The objective of this project was to create a modern, responsive, and professional website that represents the company’s services and digital identity.

The website includes essential sections such as Home, About Us, Services, and Contact. Component-based architecture was used to ensure reusability and scalability. Responsive design techniques were applied to ensure compatibility across desktops, tablets, and mobile devices. Performance optimization and clean UI design were prioritized to enhance user experience.

Website URL: https://sajilodigital.com.np

Project 2: Python WiFi Home Server

Another significant project involved the development of a Python-based home server designed to operate within a local WiFi network. The objective of this project was to understand server-client communication, local networking, and lightweight backend services.

The server allows devices connected to the same WiFi network to access services hosted on the local machine. This project helped in understanding networking concepts, request handling, and basic backend logic using Python. It also provided exposure to system-level thinking beyond frontend development.

GitHub Repository:
https://github.com/sajilodigital9000/python-server

Project 3: Color Picker Web Application

The Color Picker project is a lightweight web application developed using modern frontend technologies. The application allows users to select and preview colors easily, making it useful for designers and developers.

This project focused on user interaction, dynamic state updates, and clean UI design. The application emphasizes simplicity, responsiveness, and instant feedback, improving understanding of frontend interactivity and usability principles.

Live Demo:
https://easycolorpicker.vercel.app/

Project 4: Quiz Web Application

The Quiz Application was developed as an interactive web-based system that allows users to attempt multiple-choice questions and receive immediate feedback. The project focused on managing application state, handling user input, and dynamically rendering content.

This project improved my understanding of conditional rendering, state management, and user flow in frontend applications. Emphasis was placed on responsiveness and clarity of user interaction to ensure a smooth experience across devices.

Live Demo:
https://sajiloquiz.vercel.app/
